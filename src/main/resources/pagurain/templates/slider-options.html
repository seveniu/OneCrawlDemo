Slider

  • 基础
  • 参数说明 
$p.slider(Seletor, Options);

Options

参数 默认值 数据类型 描述
type
data-type
 "single" string 选择一种类型, single 设置一个值, double 设置一个范围
min
data-min
 10 number 设置最小值
max
data-max
 100 number 设置最大值
from
data-from
 min number 设置左手柄（或单柄）的起始位置
to
data-to
 max number 设置右手柄的起始位置
step
data-step
 1 number 设置每一步滑动的值，对于0，可以是小数
min_interval
data-min-interval
 — number 当类型为double，设置最小范围值
max_interval
data-max-interval
 — number 当类型为double，设置最大范围值
values
data-values
 [] array 设置可能的滑块值的数组，可以是数字或字符串。如果该值设置，min，max和step都无效。
from_fixed
data-from-fixed
 false boolean 禁止拖动左手柄（或者单手柄）。
from_min
data-from-min
 min number 设置左手柄拖动的最小值
from_max
data-from-max
 max number 设置左手柄拖动的最大值
to_fixed
data-to-fixed
 false boolean 禁止拖动右手柄。
to_min
data-to-min
 min number 右手柄拖动的最小值
to_max
data-to-max
 max number 设置左手柄拖动的最大值
prettify_enabled
data-prettify-enabled
 true boolean 设置长数字的可读性。 10000000 → 10 000 000
prettify_separator
data-prettify-separator
 " " string 设置长数字的可读性。 10 000, 10.000, 10-000 и т.п.
prettify
—
 null function 设置你自己的美化功能。可以是任何东西。
force_edges
data-force-edges
 false boolean 强制文字显示在控件内部
keyboard
data-keyboard
 false boolean 启用键盘操作. Move left: ←, ↓, A, S. Move right: →, ↑, W, D.
keyboard_step
data-keyboard-step
 5 number 键盘操作移动的步数。
grid
data-grid
 false boolean 启用刻度尺
grid_margin
data-grid-margin
 true boolean 设置刻度尺左右两边是否有边距
grid_num
data-grid-num
 4 number 设置刻度单元数
grid_snap
data-grid-snap
 false boolean 刻度尺单元网格滑块步（步参数）。如果启用，grid_num不会用。
hide_min_max
data-hide-min-max
 false boolean 隐藏最大值最小值的显示
hide_from_to
data-hide-from-to
 false boolean 隐藏左右手柄上的数值
prefix
data-prefix
 — string 设置值的前缀。比如将数量100设置为:$100
postfix
data-postfix
 — string 值设置后缀。比如将数值100设置为：100K
max_postfix
data-max-postfix
 — string 给最大值设置后缀
input_values_separator
data-input-values-separator
 " ; " string 设置input中两个值的分隔符。
disable
data-disable
 false boolean 禁用。
onStart
—
 null function 回调函数，滑动的时候启动。
onChange
—
 null function 回调函数，值发生变化的时候启动。
onFinish
—
 null function 回调函数，一次滑动结束的时候启动。
onUpdate
—
 null function 回调函数。触发update 或者 reset方式的时候启用。
