  • 参数说明 
$p.dataTable(Seletor, Options);

Options

参数 默认值 数据类型 描述
fnDataSource -- function 一个方法，定义dataTable显示的数据源
sClass -- string 自定义dataTable样式，class名称
aaSorting -- Array 设置默认排序字段
fnParams -- function 设置自定义请求参数
aoColumns -- Array 设置列信息
bAutoload true boolean 设置自动加载数据
bInfo true boolean 设置显示总数信息
bPaginate true boolean 设置显示分页信息
bSort true boolean 设置可以排序
bOrderNumbers false boolean 设置显示序号
sDom <"top"><"table-scrollable"rt><"bottom"pli><"clear"> string 设置dataTable布局
aLengthMenu [ [30, 50, 100], [30, 50, 100] ] Array 设置页面显示行数
oSearch -- object 设置搜索信息

Options.aoColumns 列配置

参数 默认值 数据类型 描述
bSortable true boolean 设置该字段可排序
mData -- string 设置该字段也数据对象绑定的列名
sClass -- string 设置该列的样式，class名称
bShowSummary false boolean 设置该列显示汇总数据
fnSummaryFormat false function 格式化汇总的值

Options.oSearch 搜索框配置

参数 默认值 数据类型 描述
sInput -- string 搜索文本框的选择器，例如：#txt_search
sParamName -- string 请求参数名称
fnCallback -- function 回调函数

Options.oParamName 请求参数名配置

参数 默认值 数据类型 描述
sPage page string 当前页码
sOrderColumn orderColumn string 排序列
sOrderType orderType string 排序类型,asc（升序）、desc（倒序）
sPagesize pagesize string 每页显示行数
